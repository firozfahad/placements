# Placement-Portal-Application-MAD-I-
Institutes need efficient systems to manage campus recruitment activities involving companies, students, and placement drives. Currently, many institutes rely on spreadsheets, emails, or manual processes.
# PPA-MAD Project

This repository contains the project developed for the PPA-MAD course. 
The project focuses on building an application using modern development practices and version control with GitHub.

Milestone-0 includes repository setup and initial configuration.
